# ros2-nav2-multi-robot-queue-scheduling
本项目面向多机器人共享工作站作业场景。系统通过 namespace 隔离 robot1/robot2 的 scan、odom、tf、cmd_vel 以及 Nav2 action server，实现多机器人独立定位、规划与控制。在此基础上，设计 queue_manager 调度节点，将机器人任务流程划分为前往等待点、队列等待、自动补位、进入工作站、作业执行、前往出口和任务完成等状态。针对多机器人同时访问共享资源时可能产生的工作站抢占、等待点阻塞和通道冲突问题，引入工作站锁和出口通道锁机制，保证机器人按队列顺序有序进站和出站。同时系统支持失败恢复、状态话题发布、RViz Marker 可视化和 CSV 日志统计，可用于展示调度过程并评估任务耗时、等待时间和执行状态。
